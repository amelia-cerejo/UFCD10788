Fundamentos da linguagem SQL

Uma introdução prática às bases de dados relacionais e aos fundamentos da linguagem SQL. Consultar, estruturar e gerir dados passo a passo.

Ver conteúdos Avaliação

Objetivos da formação

Identificar, consultar e gerir dados com SQL

1

Identificar bases de dados relacionais.

2

Consultar dados armazenados dentro da base de dados.

3

Criar tabelas, inserir, alterar e apagar dados.

Metodologia

Explicar, demonstrar e praticar

A formação combina explicação curta, demonstração orientada e prática acompanhada. Cada novo conceito é aplicado em exercícios progressivos, trabalhos colaborativos e desafios ligados a situações reais.

1. Conceito curto 2. Exemplo guiado 3. Exercício progressivo 4. Reflexão e melhoria

Conteúdos

Percurso de aprendizagem

1. Introdução a bases de dados

Compreender o que é uma base de dados, para que serve e como organiza informação.

2. Ambientes de bases de dados

Identificar ambientes, ferramentas e sistemas usados para trabalhar com bases de dados.

3. Terminologia de bases de dados relacionais

Reconhecer tabelas, campos, registos, chaves e relações entre dados.

4. Planeamento e desenho de bases de dados

Planear tabelas, campos e relações antes da criação da base de dados.

5. Introdução ao SQL

Conhecer a linguagem SQL e o seu papel na consulta e gestão de dados.

6. Criação de bases de dados

Criar estruturas iniciais para guardar dados de forma organizada.

7. Tabelas e integridade de dados

Criar tabelas, definir tipos de dados e respeitar regras de integridade.

8. Fundamentos de Transact-SQL

Aplicar instruções base para criar, inserir, alterar, apagar e consultar dados.

9. Filtragem e ordenação de dados

Consultar dados com critérios, filtros e ordenação de resultados.

Atividades

Prática orientada

As atividades foram pensadas para ligar a teoria à prática, permitindo aplicar progressivamente os conteúdos trabalhados ao longo da UFCD.

Cada atividade está associada a um ou mais conteúdos, para que a aprendizagem fique organizada, contextualizada e fácil de acompanhar.

Ligação aos conteúdos

As atividades serão construídas gradualmente à medida que forem sendo trabalhadas e partilhadas nas páginas dos conteúdos da UFCD.

Construção gradual

Novas atividades serão acrescentadas apenas quando forem efetivamente planeadas, realizadas ou partilhadas durante a formação.

Organização da aprendizagem

Cada atividade ficará ligada ao conteúdo onde foi trabalhada, facilitando a consulta, o acompanhamento e a revisão posterior.

Síntese das atividades reais

A secção funcionará como resumo organizado das atividades reais da UFCD, sem criar exemplos artificiais para preencher a interface.

Tarefa de grupo

Permite discutir ideias, construir soluções em conjunto, desenvolver competências de colaboração e partilhar diferentes formas de resolver problemas.

Tarefa individual

Permite consolidar a aprendizagem, aplicar os conceitos de forma autónoma e produzir evidências individuais do trabalho realizado.

Brainstorming

Atividade inicial com Mentimeter para recolher ideias, expectativas e conhecimentos prévios antes de aprofundar o primeiro tema.
